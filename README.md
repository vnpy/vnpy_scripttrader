## 说明

ScriptTrader是用于交易脚本执行的功能模块，和其他策略模块最大的区别在于其基于【时间驱动】的【同步逻辑】，也支持在命令行（Jupyter Notebook）中以REPL指令形式的进行交易操作，该模块没有回测功能。

## 安装

安装环境推荐基于3.0.0版本以上的【[**VeighNa Studio**](https://www.vnpy.com)】。

直接使用pip命令：

```
pip install vnpy_scripttrader
```


或者下载源代码后，解压后在cmd中运行：

```
pip install .
```
